## 核心理念：场景 = 世界观的物理容器

> 场景设计的终极目标不是"好看"，而是让场景本身成为叙事工具——
> 它承载情绪、定义角色所处的世界、并开放给观众进行二创。

**场景的三重功能：**
1. **叙事锚点**：每一个场景名都是 `@标签`，是 AI 生成的稳定参考点
2. **情绪容器**：场景的光影/构图/色调直接决定观众的情绪体验
3. **IP 宇宙入口**：好的场景，观众会拿去做二创（"雪山湖里"就是一个世界观符号，不是背景图）

## 场景设计的 AI 时代范式转变

| 旧范式 | 新范式 |
|--------|--------|
| 场景 = 背景板 | 场景 = 世界观前期容器 |
| 场景服务于角色 | 场景与角色共同叙事 |
| 追求精美写实 | 追求情绪辨识度 |
| 场景固定不变 | 场景开放，供观众/AI 二创 |
| 正面全景图够用 | 需要多角度、多光效变体 |

**关键认知**：设定稿中的场景描述 = 写给 AI 的提示词。场景设定越精准，后续视频生成的氛围越可控，越不需要每次重新描述。

## 场景板标准规范（主体库三板之场景板）

> 来源：白梦客知识库 `wiki/AI视频创作/Vidu/主体库大师.md`

| 字段 | 标准 |
|------|------|
| 场景名 | 简洁有辨识度，可作为 `@标签` 引用 |
| 出场集数 | 首次出场集数 |
| 场景描述 | 200字以上：时间段、地点、空间布局、光影基调、氛围 |
| 文生图描述词 | 150字以上，结构：`主体环境 + 光影参数 + 视觉风格 + 情感氛围` |

### 场景描述词结构公式

```
主体环境（地点/空间类型）
+ 时间/天气（决定自然光色温）
+ 空间逻辑（人物动线/机位可能性）
+ 光影参数（主光类型 + 介质 + 光比）
+ 氛围关键词（情绪方向）
+ 风格标签（画风/导演风格参考）
```

### 场景设计三原则

| 原则 | 说明 |
|------|------|
| **空间逻辑** | 场景布局需符合人物动线和摄影机位设置 |
| **时代/世界观特征** | 场景装饰、建筑风格需准确反映世界观背景 |
| **光影优先** | 场景光照设计需考虑时间、天气、情绪氛围需求 |

## 分镜命名与场景锚定规则

> 来源：白梦客知识库 `wiki/AI视频创作/Vidu/分镜生成系统.md`

### @标签引用规范

在分镜描述中，场景必须用 `@标签` 锚定：

```
分镜描述格式：
在【时间描述】的【@场景名】中，[@角色名] 做了什么...

示例：
在瓢泼大雨的夜晚，【@废弃剧场】中，[@小丑] 缓缓张开双臂...
```

**场景绘图提示词格式：**
```
@场景名：（室内/室外广角，建立镜头）场景的详细视觉描述...
```

### 禁止代词规则

所有分镜必须是**可独立理解的、自包含的原子化指令**。
- ❌ 禁止：「他站在那里」
- ✅ 要求：「[@角色名] 站在 [@场景名] 的中央」

## 空间比例与情绪心理学

> 空间的尺度直接操控观众的身体感知——这是比色调、光影更底层的情绪工具。

### 天花板高度 × 情绪

| 高度 | 体感 | 情绪效果 | 适用情境 |
|------|------|---------|---------|
| < 2.4m | 低矮压迫 | 焦虑、窒息、私密 | 地下室、牢房、密室逃脱 |
| 2.4m~3m | 普通住宅感 | 日常、亲切、安全 | 家庭场景、办公室 |
| 3m~5m | 开阔商业感 | 正式、公共、疏离 | 大堂、画廊、教室 |
| 5m~10m | 挑高仪式感 | 崇敬、仪式、权力 | 教堂、法院、剧院 |
| > 10m | 穹顶巨构 | 渺小、敬畏、神圣 | 大教堂、宫殿、太空站 |

### 空间宽窄 × 情绪

| 空间类型 | 体感 | 情绪效果 | AI 关键词 |
|---------|------|---------|----------|
| 狭窄走廊 < 1.2m | 挤压 | 紧张、追逐、幽闭 | `narrow corridor, claustrophobic, tight space` |
| 标准通道 1.2m~2m | 中性 | 目的性、流动 | `hallway, passage` |
| 宽敞大厅 > 5m | 空旷 | 权力、孤独、自由 | `vast hall, grand space, expansive interior` |
| 无边际空间 | 失重 | 超现实、虚无、解放 | `infinite space, boundless, void` |

### 封闭度 × 情绪

| 封闭程度 | 情绪效果 | 典型场景 |
|---------|---------|---------||
| 全封闭（无窗） | 囚禁、安全、秘密 | 地牢、保险库、暗室 |
| 半封闭（小窗/门缝） | 窥视、渴望、外面的世界 | 监狱、阁楼、病房 |
| 半开放（大窗/阳台） | 连接、过渡、犹豫 | 公寓、办公室、咖啡馆 |
| 全开放（无墙） | 自由、暴露、不安全 | 屋顶、悬崖、废墟 |

## 室内场景设计规范

### 写实场景核心检查项

> 写实风格场景的说服力 = 细节密度 × 物理逻辑一致性。AI 生成最常暴露的破绽在于空间逻辑和物件合理性。

**必查清单：**
1. **光源合理性**：光从哪来？窗户位置与光照方向是否一致？
2. **比例一致性**：家具、人物、门窗之间的比例是否合理？
3. **磨损逻辑**：高频触碰区域（门把手、扶手）应有使用痕迹
4. **重力逻辑**：物体是否稳定地放置在平面上？布料是否自然垂坠？
5. **时代一致性**：电器、家具、装饰风格是否属于同一时代？
6. **生活痕迹**：杯子上的水渍、书桌上的文件——空间要有「住过人」的感觉

### 常用室内场景模板

| 场景类型 | 空间要素 | 情绪基调 | 材质关键词 |
|---------|---------|---------|----------|
| 老式公寓 | 窄长走廊、小窗、暖色灯泡 | 怀旧、温暖、局促 | 老旧木地板、斑驳墙纸、磨损门把 |
| 现代极简公寓 | 开放式、大落地窗、简约家具 | 疏离、精致、冷感 | 白墙、抛光混凝土、不锈钢、玻璃 |
| 中式老宅/四合院 | 院落围合、木构梁柱、对联 | 礼序、传承、衰败 | 灰砖、老木、红漆柱、砖雕 |
| 日式和室 | 榻榻米、低矮桌、障子门 | 宁静、侘寂、克制 | 竹席、纸门、原木、棉麻 |
| 废弃工厂 | 高挑空间、铁架、碎玻璃 | 荒废、暗力量、重生 | 锈蚀铁、碎混凝土、油渍地面 |
| 医院/实验室 | 白色瓷砖、日光灯、不锈钢 | 无菌、冷漠、控制 | 瓷砖、不锈钢、环氧地坪 |
| 酒吧/夜店 | 低照度、霓虹、镜面反射 | 暧昧、沉醉、地下 | 黑色皮革、镜面、霓虹管 |
| 图书馆/书房 | 高书架、暖色灯、木质台面 | 知性、安全、时间沉淀 | 深色木、皮革、旧纸、黄铜台灯 |

| 施工现场 | 脚手架、塔吊、裸露结构 | 未完成、混乱、生长 | `construction site, scaffolding, crane, exposed structure` |
| 废弃游乐场 | 锈蚀旋转木马、褪色涂装 | 童年消逝、诡异 | `abandoned amusement park, rusted rides, faded paint, eerie` |
| 屋顶天台 | 城市全景、水箱、晾衣架 | 逃离、喘息、视野突变 | `rooftop, city panorama, water tanks, urban escape` |
| 地下通道 | 荧光灯、涂鸦、回声感 | 边缘、地下文化、危险 | `underground passage, fluorescent lights, graffiti, echo` |

## 空间动线与机位关系

> 场景不只是一张静态图，更是角色运动和镜头运动的舞台。设计时必须预判动线。

### 动线类型 × 叙事功能

| 动线类型 | 空间特征 | 叙事功能 | 镜头配合 |
|---------|---------|---------|---------||
| **直线型** | 走廊、桥、小路 | 目标明确的前进/追逐 | 跟拍、推轨、一点透视 |
| **环绕型** | 中庭、圆形广场 | 犹豫、仪式、对峙 | 环绕跟拍、360°运镜 |
| **迷宫型** | 老城、仓库群、森林 | 迷失、逃亡、探索 | 手持晃动、POV主观 |
| **垂直型** | 楼梯、电梯、悬崖 | 上升/坠落、权力高低 | 俯拍/仰拍、垂直推拉 |
| **停滞型** | 封闭房间、牢房 | 困境、对话、心理博弈 | 固定机位、缓慢推进 |

### 机位预设 × 空间关系

```
┌────────────────────────────────────────┐
│              全景建立镜头               │ ← 交代空间全貌
│  ┌──────────────────────────────────┐  │
│  │        中景叙事区域              │  │ ← 角色活动主区域
│  │  ┌──────────────────────────┐   │  │
│  │  │    特写情绪区域           │   │  │ ← 面部/手部/物件
│  │  └──────────────────────────┘   │  │
│  └──────────────────────────────────┘  │
│  ↑前景遮挡物                           │ ← 门框/窗框/人物肩部
└────────────────────────────────────────┘
```

**场景设计时必须回答三个问题：**
1. 全景镜头从**哪个角度**拍能最高效地建立空间感？
2. 角色的**主要活动区域**在空间的什么位置？
3. 有没有**天然的前景遮挡物**可以用？（门框、柱子、植物……）

## 建筑构成法则

> 建筑的视觉力量来自构成关系——对称/不对称、虚实、韵律、比例。
> 理解这些法则，才能在场景设计中主动**创造**空间感，而不只是复制参考图。

### 对称与不对称

| 构成方式 | 视觉效果 | 情绪语言 | 使用场景 |
|---------|---------|---------|---------|
| **轴对称** | 庄严、稳定、权威 | 秩序、神圣、压迫 | 宫殿正面、教堂中殿、法院大厅 |
| **近似对称** | 整体有序但有呼吸感 | 和谐、生活化、可亲 | 传统民居、中式院落 |
| **刻意不对称** | 动态、张力、不安定 | 现代、挑战、自由 | 解构建筑、废墟场景 |
| **局部对称+整体不对称** | 复杂但有章法 | 城市感、层次丰富 | 街景、混合建筑群 |

**场景设计原则**：
- 需要表达**秩序/权力** → 用对称
- 需要表达**自由/混乱** → 打破对称
- 最有张力的画面 = **观众期待对称但看到不对称**（如：宫殿大厅，一侧墙壁坍塌）

### 虚与实（开口与实墙）

建筑立面的「虚」（窗、门、开口）和「实」（墙体、柱子）的比例，直接控制空间的呼吸感。

| 虚实比 | 视觉效果 | 情绪 | 典型建筑 |
|--------|---------|------|---------|
| 虚 > 实（70%+开口） | 通透、轻盈、暴露 | 开放、脆弱、现代 | 玻璃幕墙办公楼、密斯·凡·德罗 |
| 虚实均衡（50/50） | 古典、稳健 | 和谐、理性 | 文艺复兴建筑 |
| 实 > 虚（70%+实墙） | 封闭、厚重、防御 | 安全、压迫、神秘 | 城堡、碉楼、监狱 |
| 全实（无窗） | 黑暗、密封 | 恐惧、禁忌 | 地牢、古墓、金库 |

### 韵律与重复

建筑中的重复元素（柱、窗、拱）形成视觉节奏——就像音乐的拍子。

| 韵律类型 | 描述 | 情绪效果 | 示例 |
|---------|------|---------|------|
| **等距重复** | 相同元素等间距排列 | 秩序、庄严、催眠 | 古典柱廊、回廊 |
| **渐变重复** | 尺寸/间距渐变 | 加速/减速、方向引导 | 透视走廊、收窄的巷道 |
| **交替重复** | A-B-A-B 交替 | 丰富但有序 | 拱窗+方窗交替的立面 |
| **打断重复** | 规律中突然打断 | 突变、重点标记 | 一排窗中一个被封死 |

**场景设计应用**：
- 走廊中等距排列的柱子 → **催眠式推进感**（适合悬疑、禅意）
- 序列中突然打断一个元素 → **视觉标点**（"这里有故事"）
- 渐变收窄 → **加速感**（追逐场景、恐惧递增）

### 比例与尺度

| 比例体系 | 原理 | 适用场景 | AI 提示词辅助 |
|---------|------|---------|------------|
| **人体尺度** | 以人体为参照物 | 日常生活场景 | `human scale, relatable proportions` |
| **纪念性尺度** | 故意放大超出人体比例 | 宫殿、神殿、权力机构 | `monumental scale, towering, dwarfing figure` |
| **微缩尺度** | 窗口、壁龛等小空间 | 亲密对话、秘密 | `intimate niche, small alcove, cozy corner` |
| **超人尺度** | 巨构、太空级 | 科幻巨型建筑 | `mega structure, impossibly large, overwhelming scale` |

**关键认知**：画面中如果没有**人或人体尺度的参照物**（家具、门），观众无法判断空间大小。
- 想让空间显大 → 在画面边缘放一个**小小的人影**
- 想让空间显亲密 → 让人物**充满画面**，压缩背景

## 空间序列设计（建筑的起承转合）

> 好的建筑就像好的电影——它引导你从入口到高潮，经历一系列精心设计的空间体验。
> 理解空间序列，才能设计出「可以走进去」的场景。

### 空间序列四段式

```
入口（起）→ 过渡（承）→ 高潮空间（转）→ 收尾/开放（合）
```

| 阶段 | 建筑空间 | 功能 | 情绪弧线 | 场景设计对应 |
|------|---------|------|---------|-----------|
| **起** | 门廊、台阶、前院 | 从外到内的心理准备 | 期待、仪式感 | 场景的建立镜头 |
| **承** | 走廊、过厅、楼梯 | 过渡，积累情绪 | 紧张递增、路径引导 | 环境气氛镜头 |
| **转** | 大厅、正殿、核心房间 | 空间高潮，视野突变 | 震撼、释放、敬畏 | 叙事核心场景 |
| **合** | 后院、阳台、望向远方 | 情绪落地或延伸 | 释然、留白、余韵 | 收尾定格镜头 |

### 经典空间序列案例

| 建筑类型 | 序列路径 | 情绪弧线 |
|---------|---------|---------|
| 中式寺庙 | 山门 → 长甬道 → 大雄宝殿 → 后院藏经阁 | 渐入佳境，层层递进 |
| 哥特教堂 | 窄门 → 高耸中殿 → 祭坛彩窗 → 侧厅静默 | 压缩→释放→敬畏→沉思 |
| 日式旅馆 | 庭院入口 → 弯折走廊 → 和室 → 窗外枯山水 | 世俗→过渡→安宁→冥想 |
| 监狱 | 铁门 → 层层铁栏 → 牢房 → 无出路 | 递进压迫，无释放 |
| 摩天大楼大堂 | 玻璃门 → 挑高大厅 → 电梯 → 顶层视野 | 突然放大→封闭→极致开放 |

**场景设计应用**：
- 设计场景时思考：**角色到达这里之前经过了什么空间？之后去哪？**
- 空间序列中的**对比越强，情绪冲击越大**（从极窄走廊直接进入穹顶大厅）
- AI 出图时可以拆解为：入口图 → 过渡图 → 高潮图 → 收尾图，形成叙事链

## 建筑与光的关系（非影视打光，而是空间如何塑造光）

> 影视打光是「把光放在空间里」；建筑设计光是「用空间本身创造光」。
> 理解建筑如何捕获和塑造光，才能设计出光影自然合理的场景。

### 建筑采光方式 × 情绪

| 采光方式 | 原理 | 情绪效果 | 经典案例 | AI 提示词 |
|---------|------|---------|---------|----------|
| **侧窗光** | 光从墙面开口水平射入 | 日常、真实、方向感 | 维米尔画中的荷兰窗 | `side window light, directional natural light` |
| **高窗/天窗光** | 光从高处垂直落下 | 神圣、恩赐、仪式 | 万神殿穹顶天窗 | `skylight from above, zenith light, cathedral clerestory` |
| **缝隙光** | 光从窄缝/裂口渗入 | 希望、窥视、戏剧性 | 安藤忠雄光之教堂 | `light through narrow slit, beam of light in darkness` |
| **反射光** | 光经水面/白墙反射进入 | 柔和、漂浮、梦幻 | 威尼斯水面反射 | `reflected light from water, bounced light, soft indirect` |
| **遮蔽光** | 光被格栅/百叶/树叶过滤 | 斑驳、韵律、层次 | 阿拉伯穆什拉比耶窗 | `filtered light through lattice, dappled shadows, screen pattern` |
| **门洞框光** | 光从门洞射入暗室 | 召唤、出路、戏剧 | 胡金铨武侠的门洞剪影 | `doorway light, silhouette in doorframe, light from beyond` |

### 暗适应与光适应

人眼从亮到暗或从暗到亮需要适应时间——建筑大师利用这一点创造**空间惊喜**。

```
经典手法：
暗走廊（压缩、适应暗环境）
    → 突然进入明亮大厅（瞬间视觉爆炸）
    = 情绪上的「豁然开朗」

反向手法：
明亮室外 → 窄暗入口 → 逐渐适应
    = 仪式化的「进入另一个世界」
```

**场景设计应用**：
- 如果前一场景是暗调 → 下一场景用高亮度，制造**视觉和情绪的双重冲击**
- 窗户/门洞 = 天然的**画框**和**光源**，是场景设计的黄金元素

## 立面设计语言（建筑的"表情"）

> 建筑立面就像人脸——它传达个性、时代、阶级、情绪。
> 设计场景时，了解立面语言可以让空间的「第一印象」精确可控。

### 立面元素 × 叙事功能

| 立面元素 | 叙事信号 | 写实场景设计建议 |
|---------|---------|---------------|
| **大门** | 入口=邀请/拒绝 | 豪华大门=权力；生锈铁门=废弃；虚掩的门=悬念 |
| **窗户** | 内外关系=看/被看 | 亮窗=有人在；暗窗=空置；被封的窗=秘密/恐惧 |
| **阳台** | 展示/观看=公共的私密 | 晾晒衣物=生活；空阳台=孤独；倾斜护栏=危险 |
| **屋顶** | 轮廓线=第一印象 | 尖屋顶=哥特/恐怖；平屋顶=现代；坍塌=废弃 |
| **台阶** | 高差=权力/仪式 | 长阶梯=庄严；残破台阶=衰败；无台阶直入=亲和 |
| **招牌/标识** | 功能/时代/文化 | 霓虹招牌=都市夜；手写牌=传统；电子屏=未来 |
| **管线/空调外机** | 时代真实感 | 写实场景必备——外露管线=真实生活感，干净=刻意或新建筑 |
| **涂鸦/张贴物** | 文化/阶层信号 | 涂鸦=底层/反叛；竞选海报=时代；寻人启事=悬念 |

### 立面的时间叙事

同一栋建筑的立面，经历时间会产生**层叠痕迹**：

```
原始层：建造时的材料、风格、色彩
  ↓
第一层改造：换窗、加层、改门
  ↓
第二层叠加：招牌、广告、涂鸦
  ↓
第三层侵蚀：风化、污渍、植物入侵
  ↓
= 一面墙讲述一个地方的历史
```

**场景设计应用**：
- 新建筑 = 干净立面 → 未来/乌托邦/无人格
- 老建筑 = 叠加痕迹 → 故事感/历史/人文
- 废弃建筑 = 被自然侵蚀 → 末日/遗忘/时间无情

## 环境叙事（用空间讲故事）

> 好的场景不需要台词就能讲故事——通过物件摆放、磨损痕迹、空间状态来暗示「这里发生过什么」。

### 环境叙事五层模型

| 层 | 内容 | 示例 |
|----|------|------|
| **第1层：功能** | 这个空间是做什么的 | 厨房、办公室、教堂 |
| **第2层：居住者** | 谁在这里生活/工作 | 书架上的书暗示主人的身份 |
| **第3层：状态** | 当前发生了什么 | 桌上未完成的饭菜=匆忙离开 |
| **第4层：历史** | 过去发生过什么 | 墙上褪色的照片=失去的人 |
| **第5层：预兆** | 即将发生什么 | 窗外乌云密布=风暴将至 |

### 环境叙事物件清单

| 物件类型 | 叙事功能 | 示例 |
|---------|---------|------|
| **日常物件** | 暗示居住者性格 | 整齐排列的鞋=强迫症；散乱的酒瓶=堕落 |
| **时间物件** | 暗示时间流逝 | 日历停在某日=创伤；积灰厚度=废弃年限 |
| **伤痕物件** | 暗示冲突发生 | 墙上的拳头印=暴怒；刀痕=暴力 |
| **缺席物件** | 用空缺暗示存在 | 餐桌上多出一副碗筷=等待/思念 |
| **入侵物件** | 暗示外力介入 | 窗上的弹孔=战争；封条=查封 |
| **自然入侵** | 暗示被遗弃 | 地板长出的青草=长期无人 |

**场景设计原则**：
- 每个场景至少放置 **1-2 个叙事物件**
- 物件要「自然」而非「刻意」——观众应该感觉是自己发现的
- 写实场景：磨损和污渍要符合**使用逻辑**（高频触摸区域先磨损）

## 城市空间设计（街景与天际线）

> 当场景从单一建筑扩展到城市级别，需要理解城市肌理、街道比例和天际线构成。

### 街道比例（D/H比）

街道的宽度（D）与两侧建筑高度（H）的比值，直接决定街道的空间感受：

| D/H 比 | 体感 | 情绪 | 典型场景 |
|--------|------|------|---------|
| < 0.5 | 极窄峡谷感 | 压迫、幽闭、紧张 | 中世纪老城巷道、九龙城寨 |
| 0.5~1 | 舒适的围合感 | 亲密、安全、人情味 | 欧洲传统街道、江南水乡 |
| 1~2 | 开阔但有界 | 正式、商业、都市 | 城市主街、林荫大道 |
| 2~3 | 宽阔有距离感 | 疏离、现代、速度 | 现代城市大马路 |
| > 3 | 空旷无遮蔽 | 暴露、寂寥、荒凉 | 新区空城、末日场景 |

### 天际线构成

| 天际线类型 | 视觉特征 | 情绪 | AI 提示词 |
|---------|---------|------|----------|
| **锯齿型** | 高低错落、丰富轮廓 | 活力、混乱、有机 | `irregular skyline, varied building heights, organic cityscape` |
| **平坦型** | 统一高度 | 压抑、控制、单调 | `flat uniform skyline, identical buildings, monotonous` |
| **尖刺型** | 少量超高建筑突出 | 权力、野心、现代 | `skyline with dominant towers, skyscraper piercing sky` |
| **穹顶型** | 圆顶/塔楼主导 | 宗教、古典、文化 | `dome-dominated skyline, minarets, historic skyline` |
| **自然融合型** | 建筑嵌入山体/树林 | 和谐、隐遁、世外 | `buildings nestled in mountains, architecture merged with nature` |

### 城市肌理（鸟瞰质感）

| 肌理类型 | 特征 | 暗示 |
|---------|------|------|
| **有机肌理** | 不规则道路、自发生长 | 历史悠久、自然形成 |
| **网格肌理** | 正交棋盘、理性规划 | 殖民、现代规划、控制 |
| **放射肌理** | 从中心点向外辐射 | 权力中心、首都 |
| **碎片肌理** | 断裂不连续 | 战争/灾后重建 |

**场景设计应用**：
- 设计城市场景时，先画**鸟瞰草图**确定肌理类型
- 天际线是城市场景的「第一印象」，相当于角色设计中的「剪影辨识度」
- 街道的 D/H 比 = 最简单也最有效的空间情绪控制工具

## 构图法则：三层景深系统

> 场景构图的核心是**三层景深**——前景、中景、背景各有叙事功能。
> 三者的比例组合决定了画面的情绪方向，可灵活搭配任何风格。

### 三层景深总览

| 层 | 功能 | 比例范围 | 核心作用 |
|----|------|---------|---------|
| **前景层** | 框架/遮挡/层次 | 画面 10%~35% | 创造深度感和窥视感 |
| **中景层** | 主体/叙事核心 | 画面 20%~60% | 承载角色和动作 |
| **背景层** | 留白/环境/延伸 | 画面 40%~80% | 提供情绪呼吸空间 |

### 前景遮挡

创造画面层次感，是 AI 生成中提升「电影感」的最有效手法。

| 参数 | 数值 |
|------|------|
| 遮挡比例 | 10%~35% 画面面积 |
| 虚化程度 | 30%~100% |
| 光圈等效 | f/1.4~f/3.5 |

**常用前景物体：**
- 自然：竹子、树枝、芦苇、荷叶、雾气、花瓣
- 建筑：门框、窗框、屋檐、纱帘、灯笼、柱子
- 人物局部：肩部、后脑勺、侧脸轮廓
- 现代：玻璃、栏杆、霓虹招牌、车窗
- 特殊：雨滴（镜头面）、蛛网、烟雾缭绕

**遮挡比例与效果：**

| 遮挡比例 | 效果 | 适用情境 |
|---------|------|---------|
| 10%~15% | 自然、干净、轻微框架感 | 日常叙事、清新氛围 |
| 20%~25% | 层次感、空间感、氛围 | 电影感通用构图 |
| 30%~35% | 压迫感、框架感、窥视感 | 悬疑、监视、隐秘 |

**AI 提示词速查：**
```
# 标准前景遮挡
foreground [物体] obstruction, [比例]% frame coverage, shallow DOF

# 自然场景
foreground bamboo stalks obstruction, 25% frame, shallow DOF
foreground cherry blossoms, soft bokeh, 15% frame

# 窥视/监视感
double frame composition, foreground + doorway frame
peeping through window, voyeuristic framing

# 城市写实
foreground rain-streaked glass, 20% frame, blurred
foreground iron railing, shallow depth of field
```

### 极简留白（负空间）

大面积负空间——东方美学「计白当黑」，西方极简主义同样推崇。
关键不在于留白本身，而在于**主体与空白的比例关系**。

| 留白比例 | 情绪效果 | 适用情境 |
|---------|---------|---------|
| 40%~50% | 呼吸感、舒适、开放 | 日常、舒缓叙事 |
| 50%~60% | 沉思、内省、安静 | 独白、冥想 |
| 60%~70% | 孤独、疏离、仪式感 | 告别、孤立、哲思 |
| 70%~80% | 极简、压迫、虚无 | 极端情感、高端品牌 |

**留白类型：**

| 类型 | 描述 | AI 提示词 |
|------|------|----------|
| 天空留白 | 主体在底部，大面积天空 | `figure at bottom, vast sky above, low horizon` |
| 地面留白 | 主体偏上，大面积地面 | `figure in upper third, expansive ground, high angle` |
| 侧向留白 | 主体偏一侧，朝向空白方向 | `figure on left, looking into empty space, rule of thirds` |
| 全向留白 | 主体居中但极小 | `tiny figure in vast space, extreme wide shot, isolated` |

**AI 提示词速查：**
```
# 通用极简留白
negative space [留白比例]%, minimalist composition
figure on [位置], [占比]% frame

# 空旷感
figure in vast empty space, negative space 70%
long static take, minimalist

# 高端品牌
product 25% frame, negative space 70%
minimalist, solid background, rim lighting
```

### 情绪元素点缀

用**单一细节物体**传达大情绪——越小越具体，越有力量。

| 技法 | 描述 | 示例 |
|------|------|------|
| 微距细节 | 极近距离捕捉微小元素 | 水滴、烟雾、尘埃、指尖触碰 |
| 色彩点缀 | 小面积（5%~10%）高饱和度色彩 | 灰色废墟中的一朵红花 |
| 动态元素 | 在静态构图中加入微动 | 飘动的帘幕、飞舞的萤火虫 |
| 光影锚点 | 一束光指向情绪焦点 | 一束侧光打在遗留的照片上 |
| 缺席暗示 | 用物件暗示不在场的人 | 桌上多出的一杯咖啡、空荡的秋千 |

### 构图风格融合参考

> 以下是不同导演/风格的构图倾向，作为融合创作的**参考素材**，
> 实际应用中应自由组合，不必拘泥于单一风格。

| 风格来源 | 构图偏好 | 可提取的技法 |
|---------|---------|-----------|
| 胡金铨 | 前景遮挡 30% + 深景深 + 禅意留白 | 竹林/门框的遮挡层次 |
| 北野武 | 宽阔比例、人物极小、大面积天空/海 | 极端负空间的孤独感 |
| 李安 | 细节物件传情、克制的中景 | 用一个物件讲整部戏的情感 |
| 王家卫 | 残缺构图，只拍局部、倾斜角度 | 碎片式框取、不完整即美 |
| 韦斯·安德森 | 极致对称、粉彩色块、正面平视 | 强迫症级对称的仪式感 |
| 塔可夫斯基 | 缓慢横移、水面反射、自然长镜头 | 诗意时间感、水作为镜面 |
| 今敏 | 虚实交错、空间折叠、match cut | 梦境与现实的无缝切换 |
| 是枝裕和 | 窗光侧照、家庭空间、低机位 | 日常生活的诗意距离 |
| 大卫·芬奇 | 精密对称+冷调、顶光、压迫感 | 强迫症空间的不安 |
| 宫崎骏 | 层叠丰富的背景、动态前景、飞翔视角 | 手绘感的深度层次 |

**融合示例**：
```
胡金铨的前景遮挡 + 王家卫的碎片化框取 + 赛博朋克的霓虹色彩
= 雨夜霓虹巷道中，透过铁栏杆的缝隙看到角色的局部剪影
```

### 构图应用原则

1. **层次优先**：先建立前景-中景-背景三层
2. **留白克制**：宁多勿少，保持画面呼吸感
3. **情绪物体**：选择有象征意义的细节
4. **风格自由融合**：从不同来源提取技法组合，创造你自己的视觉语言
5. **真实感检查**：前景材质是否真实自然？散景是否不规则？光影是否符合物理逻辑？

## 场景光影系统

> 光影是场景情绪的底层操作系统。以下是通用的光影类型和组合方式，
> 可根据世界观需要自由搭配，不限于任何特定美学流派。

### 五种基础光影类型

| 光影类型 | 原理 | 情绪效果 | AI 提示词 |
|---------|------|---------|----------|
| **侧逆光** | 光从主体侧后方射入，勾勒轮廓 | 戏剧感、神秘、诗意 | `side-backlighting, rim light, dramatic silhouette` |
| **顶光** | 光从正上方直射 | 审判感、压迫、神圣 | `overhead lighting, top-down light, harsh shadows under eyes` |
| **自然散射光** | 阴天/漫射，无明显方向 | 日常、真实、柔和 | `overcast natural light, soft diffused, no harsh shadows` |
| **单一人造光源** | 一盏灯/一支蜡烛/一个屏幕 | 孤独、聚焦、亲密 | `single light source, pool of light in darkness` |
| **霓虹/人工色光** | 多色人造光源，漫射混合 | 都市、暧昧、赛博 | `neon glow, colored artificial light, diffused neon` |

### 光影参数速查

| 参数 | 数值范围 | 说明 |
|------|---------|------|
| 光源角度 | 0°~180° | 0°=正面；90°=侧面；180°=完全逆光 |
| 光源高度 | 0°~90° | 0°=水平；45°=标准；90°=顶光 |
| 光比 | 1:1~8:1 | 越高越戏剧，越低越柔和 |
| 色温 | 2000K~8000K | 低=暖黄；高=冷蓝 |

### 介质效果（烟雾/雨/尘）

光需要**介质**才能被看到。介质决定了光的质感：

| 介质 | 视觉效果 | 情绪 | AI 提示词 |
|------|---------|------|----------|
| 薄雾 5%~10% | 隐约可见的光束 | 诗意、仙境、清晨 | `light mist, subtle fog, gentle atmosphere` |
| 中雾 10%~20% | 明显的体积光/丁达尔 | 神秘、怀旧、仪式 | `volumetric light, god rays, visible light beams` |
| 浓雾 20%~40% | 能见度低，轮廓模糊 | 恐惧、迷失、末日 | `heavy fog, low visibility, obscured, eerie mist` |
| 雨 | 光被打散/地面反射 | 浪漫、悲伤、净化 | `rain, wet surface reflections, scattered light` |
| 尘埃 | 光束中的粒子 | 废弃、时间感、荒凉 | `dust particles in light beam, abandoned atmosphere` |
| 雪 | 反射增强，画面提亮 | 纯洁、死寂、冷酷 | `snow reflecting light, bright cold atmosphere` |

### 霓虹/人工色光

| 色光类型 | 色温 | 适用场景 |
|---------|------|---------|
| 暖黄/琥珀 | 2800K~3000K | 餐厅、酒吧、怀旧、温馨 |
| 冷白/蓝 | 4000K~6500K | 办公、医院、都市疏离 |
| 粉红/品红 | — | 爱情、夜晚、暧昧 |
| 绿/青 | — | 赛博朋克、神秘、诡异 |
| 混合多色 | — | 夜市、游乐场、迷幻 |

### 光影组合示例（按场景选择）

| 世界观场景 | 光影组合 | AI 提示词模板 |
|---------|---------|-------------|
| 古代竹林 | 侧逆光 + 薄雾 | `side-backlighting through bamboo, light mist, volumetric` |
| 废弃仓库 | 单一窗光 + 尘埃 | `single shaft of light through broken window, dust particles` |
| 赛博朋克街道 | 霓虹多色 + 雨面反射 | `neon-lit street, rain reflections, colorful light bleeding` |
| 雪山神殿 | 顶光 + 雪反射 | `overhead light, snow reflecting, cold sacred atmosphere` |
| 日常咖啡馆 | 侧窗自然光 + 散射 | `window light, warm interior, soft natural, contemplative` |
| 地下通道 | 荧光灯 + 潮湿反光 | `fluorescent light, wet floor reflections, underground` |

## 分镜结构：场景如何服务于叙事节奏

### 1+3 叙事矩阵（非对话场景）

每个基础动作单元，生成 4 个分镜——从不同维度呈现同一场景：

| 镜头 | 功能 | 场景侧重 |
|------|------|---------|
| 镜头1 — 标准叙事镜 | 客观呈现事件 | 交代场景全貌，建立空间关系 |
| 镜头2 — 情感聚焦镜 | 角色内心/微表情 | 场景细节作为情绪锚点 |
| 镜头3 — 动态冲击镜 | 速度、力量、视觉冲击 | 场景的动态元素（雨、风、烟） |
| 镜头4 — 氛围意境镜 | 环境、光影与角色互动 | 场景的情绪语言（逆光剪影、雾气等） |

**示例（废弃剧场场景）：**
```
【核心场景绘画提示词】
@废弃剧场：（室内广角，建立镜头）
一座废弃的巴洛克风格大剧院，穹顶破了大洞，
雨水和月光从中洒落。舞台上空无一物，
只有厚重的红色幕布在穿堂风中微微飘动。

【分镜1 — 标准叙事镜】
在月光洒落的[@废弃剧场]舞台中央，[@角色名]缓缓张开双臂，
像是在拥抱整个空无一人的剧场。

【分镜4 — 氛围意境镜】
在深夜的[@废弃剧场]中，镜头从舞台后方逆光拍摄，
[@角色名]张开双臂的剪影被穹顶洒下的月光勾勒出来，
显得孤独而神圣。
```

### 九宫格分镜结构（15 秒片段）

每组 9 个分镜 = 1 个 15 秒片段，场景功能对应结构位置：

```
┌──────────────┬──────────────┬──────────────┐
│ 1-开场（场景） │ 2-环境（气氛） │ 3-情绪（角色） │  ← 铺垫（0~5s）
├──────────────┼──────────────┼──────────────┤
│ 4-转折（冲突） │ 5-高潮（核心） │ 6-爆发（动作） │  ← 核心（5~10s）
├──────────────┼──────────────┼──────────────┤
│ 7-回落（余韵） │ 8-情感（细节） │ 9-定格（余韵） │  ← 收尾（10~15s）
└──────────────┴──────────────┴──────────────┘
```

**场景镜头在九宫格中的位置：**
- `分镜1` = 场景建立镜头（广角/全景）
- `分镜2` = 环境气氛镜头（光影/色调呈现）
- `分镜4/5` = 场景与角色动作的核心互动
- `分镜9` = 场景的情绪收尾（常用：远景拉远 / 环境特写定格）

## 速查：场景设计关键决策点

| 决策 | 推荐做法 |
|------|---------|
| 场景需要传达什么情绪？ | 先定情绪方向，再选光影/构图参数 |
| 光影选哪种？ | 从五种基础光影类型中选择，按世界观自由组合 |
| 构图层次怎么建立？ | 前景遮挡 + 极简留白 + 情绪细节点缀，比例自由 |
| 场景描述给 AI 多少字？ | 文生图描述词 150 字以上，含：空间/材质/光影/色调/情感 |
| 场景在分镜中如何引用？ | 用 `@场景名` 标签，禁止代词，保持原子化 |
| 如何防止场景 AI 输出随机性？ | 先出场景板设定图，锁定后作为参考图输入 |
