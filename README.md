## 这是什么

一份可直接加载到 AI Agent 的**场景设计技能文件**，覆盖从理念到落地的完整工作流。

> **核心主张**：场景不是背景板，而是世界观的物理容器——它承载情绪、定义角色所处的世界、并开放给观众进行二创。

## 快速开始

### 1. 安装到 AI Agent

```bash
# Claude Code / ECC
cp SKILL.md ~/.claude/.agents/skills/scene-design/SKILL.md

# 或复制到其他支持 skill 的 AI 工具
cp SKILL.md <your-agent-skills-dir>/scene-design/SKILL.md
```

### 2. 使用场景

当你在对话中提到以下关键词时，AI 会自动加载此技能：

| 关键词 | 场景 |
|--------|------|
| 场景设计、设计场景 | 新场景开发 |
| 分镜、分镜头 | 场景分解与镜头规划 |
| 构图、光影 | 视觉语言设计 |
| 场景诊断 | 现有场景问题分析 |
| @标签、场景名 | 场景锚定与引用 |

### 3. 核心工作流

```
确定叙事需求
    ↓
场景诊断（这个场景要完成什么叙事任务？）
    ↓
空间特性设计（需要什么空间特性？）
    ↓
视觉语言转化（如何转化为视觉语言？）
    ↓
生成场景板（场景描述词 + 分镜规划）
```

### 构图三层系统

```
┌─────────────────────────────────────────────────┐
│  前景遮挡 10%-35%    │  胡金铨    │  层次/窥视感  │
├─────────────────────────────────────────────────┤
│  主体                 │            │  叙事核心      │
├─────────────────────────────────────────────────┤
│  留白 40%-80%         │  北野武    │  情绪呼吸空间   │
└─────────────────────────────────────────────────┘
```

### 光影核心配方

```
侧逆光 45°-60° + 烟雾 30%-50% + 光比 2:1-4:1 + 负空间 50%-70%
```

### 七情速查

| 情绪 | 色调 | 光影 | 场景元素 |
|------|------|------|---------|
| 惧 | 冷蓝白 | 顶光+烟雾 | 封闭走廊、无尽楼梯 |
| 怒 | 高饱和红 | 侧光+顶光 | 挤压感空间 |
| 哀 | 大地+苍青 | 侧逆+自然光 | 脚印消失处 |
| 喜 | 琥珀+暖金 | 柔光+漫射 | 温暖角落 |
| 恶 | 深紫黑 | 底光+阴影 | 反射物少 |
| 爱 | 琥珀+苍青 | 侧逆对比 | 暧昧边界光 |
| 欲 | 深红+金 | 聚光+暗部 | 焦点明确 |

## 场景诊断工作流

### 三层分析框架

```
叙事需求
    ↓
第一层：这个场景要完成什么叙事任务？
    ↓
第二层：完成这个任务需要什么空间特性？
    ↓
第三层：这些特性如何转化为视觉语言？
```

### 叙事任务分类

| 叙事任务 | 本质问题 | 例子 |
|---------|---------|------|
| 建立世界观 | 这个世界是什么样的？ | 展示外星城市 |
| 揭示角色 | 这是一个什么样的人？ | 展示角色住所 |
| 建立关系 | 角色之间是什么关系？ | 两人在什么空间对话 |
| 制造张力 | 故事在积累什么？ | 暴风雨前的平静 |
| 释放情绪 | 观众应该感受到什么？ | 孤独、温暖、压迫 |
| 转折标记 | 故事发生了什么变化？ | 同一个房间但变了 |
| 收束主题 | 故事在说什么？ | 主题物件再次出现 |

## @标签系统

场景设计使用 `@标签` 语法实现场景锚定：

```markdown
# 正确用法
在【@废弃剧场】中，[@李逍遥]站在舞台中央

# 错误用法
他在那个废弃的剧场里站着
```

**@标签规则：**
- 每个场景名都是 `@标签`，是AI生成的稳定参考点
- 禁止使用代词（他/她/它/这个/那个）
- 所有指代必须原子化

## 速查清单

| 决策 | 检查项 |
|------|--------|
| 构图 | 是否符合三层景深（前景10-35%+主体+留白40-80%）？ |
| 光影 | 是否符合光影配方（侧逆光+烟雾+光比2:1-4:1）？ |
| 情绪 | 是否明确传达目标情绪（参考七情参数）？ |
| 叙事 | 场景是否完成预定的叙事任务？ |
| @标签 | 是否使用@标签锚定而非代词？ |
