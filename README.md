# AI 场景设计技能 (Scene Design Skill)

AI时代动画/漫剧前期场景设计方法论。

## 覆盖内容

- 场景的世界观功能与情绪引导
- 构图法则（前景遮挡 × 极简留白 × 情绪点缀）
- 光影公式（侧逆光+烟雾 × 霓虹漫射）
- 七情 × 场景视觉参数速查
- 分镜结构（1+3叙事矩阵 / 九宫格）
- 场景板规范与输出模板
- AI出图工作流

## 知识来源

- 皓哥内门培训逐字稿（角色与场景设计方法论）
- 白梦客知识库：构图法则、光影公式、分镜生成系统、情绪引导方法论

## 使用方式

将 `SKILL.md` 放入你的 AI agent 的 skills 目录即可。
