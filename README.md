## 🧭 这是什么

一份可直接加载到 AI Agent（Claude Code / Cursor / Windsurf 等）的**场景设计技能文件**，覆盖从理念到落地的完整工作流。

> **核心主张**：场景不是背景板，而是世界观的物理容器——它承载情绪、定义角色所处的世界、并开放给观众进行二创。

### 适用场景

| 场景 | 说明 |
|------|------|
| 🎥 漫剧 / 短剧制作 | 前期场景设定、分镜场景规划 |
| 🤖 AI 视频生成 | 为 Vidu / 即梦（Seedance）/ 可灵等模型提供精准场景提示词 |
| 🎨 概念设计 | 世界观场景的视觉风格定义 |
| 📋 甲方提案 | 场景板输出、导演手记生成 |

## 🚀 快速开始

### 1. 作为 AI Agent Skill 使用

将 `SKILL.md` 放入你的 skills 目录：

```bash
# Claude Code / ECC
cp SKILL.md ~/.claude/.agents/skills/scene-design/SKILL.md

# 或任何支持 skill 加载的 AI 工具
cp SKILL.md <your-agent-skills-dir>/scene-design/SKILL.md
```

之后当你在对话中提到「场景设计」「设计场景」等关键词时，AI 会自动加载此技能协助你。

### 2. 作为独立参考手册

直接阅读 [`SKILL.md`](SKILL.md) 即可，所有方法论、参数表、提示词模板均为即用型。

## 🎨 核心亮点

### 构图三层系统

```
┌─────────────────────────────────────────────────┐
│  前景遮挡 10%-35%    │  胡金铨    │  层次 / 窥视感  │
├─────────────────────────────────────────────────┤
│  主体               │           │  叙事核心      │
├─────────────────────────────────────────────────┤
│  留白 40%-80%       │  北野武    │  情绪呼吸空间   │
└─────────────────────────────────────────────────┘
```

### 光影核心配方

```
侧逆光 45°-60° + 烟雾 30%-50% + 光比 2:1-4:1 + 负空间 50%-70%
```

### 七情速查（示例）

| 情绪 | 色调 | 光影 | 场景元素 |
|------|------|------|---------|
| 惧 | 冷蓝白 | 顶光+烟雾 | 封闭走廊、无尽楼梯 |
| 哀 | 大地+苍青 | 侧逆+自然光 | 脚印消失处、拉上的窗帘 |
| 爱 | 琥珀+苍青 | 侧逆对比 | 暧昧边界光、反射水面 |

> 完整七情参数见 [SKILL.md](SKILL.md#七情与场景视觉参数速查)
